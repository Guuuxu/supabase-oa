# 使用手册功能说明

## 概述
为九头鸟人事托管签署系统创建了完整详细的使用手册，并集成到系统管理菜单中。

## 功能特性

### 1. 完整的内容结构
使用手册包含以下7个主要章节：

#### 1.1 系统概述
- 系统介绍：系统的定位和主要用途
- 主要功能：列出系统的核心功能模块
- 用户角色说明：详细说明超级管理员、主管、员工三种角色的权限和职责

#### 1.2 快速入门
- 登录系统：登录步骤和注意事项
- 界面导航：介绍系统界面的各个部分
- 基本操作：查看、添加、编辑、删除数据的通用操作方法

#### 1.3 客户管理
- 功能说明：客户管理模块的用途
- 添加客户：详细的添加客户步骤，包括基本信息和服务信息
- 筛选客户：支持的各种筛选方式
- 查看客户详情：详情页面包含的信息

#### 1.4 员工管理
- 功能说明：员工管理模块的用途
- 添加员工：详细的添加员工步骤，包括基本信息和劳动合同信息
- 员工状态管理：6种员工状态的说明（在职、请假、休假、待岗、出差、离职）
- 查看员工详情：详情页面包含的信息
- 批量导入员工：Excel批量导入的步骤

#### 1.5 文书管理
- 文书模板库：公司模板和通用模板库的区别
- 文书分类：详细列出5大类文书（入职管理、在职管理、离职管理、薪酬管理、证明开具）及其子类
- 发起签署：详细的签署发起流程，包括跨模板多选、签署流程说明
- 查看签署状态：4种签署状态的说明
- 签署数据调取：数据统计和导出功能

#### 1.6 薪资管理
- 工资结构模板：设置通用工资结构模板的步骤
- 上传工资表：Excel工资表上传和自动拆分的流程
- 考勤管理：考勤数据录入和管理
- 薪酬签署管理：工资条和考勤表的签署流程，支持批量操作

#### 1.7 系统配置
- 用户管理：添加、编辑、停用用户，分配角色
- 角色设置：自定义角色和权限配置，详细列出11种权限项
- 系统设置：系统参数配置，包括提醒设置、API配置等

#### 1.8 常见问题
- 登录相关：忘记密码、权限问题
- 操作相关：批量导入、签署短信、撤回签署、修改工资表、查看签署历史
- 数据相关：数据导出、批量删除、数据恢复
- 技术支持：联系方式和工作时间

### 2. 交互式导航
- **左侧目录**：显示所有章节，点击可快速跳转
- **搜索功能**：支持按章节标题搜索，快速定位内容
- **高亮显示**：当前选中的章节高亮显示
- **图标标识**：每个章节都有对应的图标，便于识别

### 3. 响应式设计
- **桌面端**：左右分栏布局，目录和内容同时显示
- **移动端**：自适应布局，确保在小屏幕上也能良好显示
- **滚动区域**：内容区域独立滚动，不影响目录导航

### 4. 视觉设计
- **清晰的层级结构**：使用不同字号和颜色区分标题层级
- **卡片式布局**：重要信息使用卡片展示，突出重点
- **列表和编号**：使用有序列表和无序列表，结构清晰
- **色彩区分**：使用背景色区分不同类型的内容块

### 5. 内容特点
- **详细的步骤说明**：每个操作都有详细的步骤指导
- **实用的提示信息**：提供注意事项和最佳实践
- **完整的功能覆盖**：涵盖系统的所有主要功能模块
- **常见问题解答**：收集用户常见问题并提供解答

## 技术实现

### 组件结构
```
UserManualPage
├── 页面标题和描述
├── 左侧目录卡片
│   ├── 搜索框
│   └── 章节列表（可滚动）
└── 右侧内容卡片
    └── 章节内容（可滚动）
```

### 数据结构
```typescript
interface Section {
  id: string;           // 章节ID
  title: string;        // 章节标题
  icon: React.ReactNode; // 章节图标
  content: React.ReactNode; // 章节内容
}
```

### 状态管理
- `searchKeyword`：搜索关键词
- `activeSection`：当前激活的章节ID
- `filteredSections`：过滤后的章节列表

### UI组件使用
- `MainLayout`：主布局组件
- `Card`：卡片容器
- `ScrollArea`：滚动区域
- `Input`：搜索输入框
- `Button`：章节导航按钮
- Lucide图标：各种功能图标

## 菜单集成

### 路由配置
- 路径：`/user-manual`
- 组件：`UserManualPage`
- 位置：顶部导航栏

### 顶部导航栏
- 按钮样式：outline变体，小尺寸
- 图标：BookOpen
- 文本：使用手册（桌面端显示，移动端隐藏）
- 位置：顶部导航栏左侧区域
- 权限：无权限限制，所有用户均可访问

### 设计说明
- 使用Button组件的asChild属性配合Link实现导航
- 响应式设计：移动端只显示图标，桌面端显示图标+文字
- 使用outline变体使按钮更加醒目
- 位置显眼，方便用户随时访问

## 使用方式

### 访问使用手册
1. 登录系统
2. 在顶部导航栏左侧找到"使用手册"按钮
3. 点击按钮进入使用手册页面

### 浏览内容
1. 在左侧目录中点击章节标题
2. 右侧内容区域自动显示对应章节内容
3. 可以上下滚动查看完整内容

### 搜索章节
1. 在左侧目录顶部的搜索框中输入关键词
2. 目录自动过滤显示匹配的章节
3. 点击搜索结果快速跳转
