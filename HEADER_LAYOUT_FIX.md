# 薪酬签署页面 - 优化标题栏布局

## 修改说明
缩短公司选择器的宽度，使"签署状态统计"标题和公司选择器能在一行显示，不换行。

## 问题分析

### 修改前的问题
- CompanySelector组件内部使用了`w-full`类名，导致它占据了所有可用宽度
- 在flex布局中，当标题和选择器都试图占据空间时，会导致换行
- 没有明确限制选择器的宽度

### 解决方案
1. 给CompanySelector外层包裹一个固定宽度的div容器
2. 设置容器宽度为`w-64`（256px），足够显示公司名称
3. 给标题添加`whitespace-nowrap`，防止标题文字换行
4. 给flex容器添加`gap-4`，确保标题和选择器之间有适当间距

## 修改内容

### 文件位置
`src/pages/SalarySignaturesPage.tsx` 第759-768行

### 修改前代码
```tsx
<CardHeader>
  <div className="flex items-center justify-between">
    <CardTitle>签署状态统计</CardTitle>
    <CompanySelector
      companies={companies}
      value={selectedCompanyId}
      onValueChange={setSelectedCompanyId}
    />
  </div>
</CardHeader>
```

### 修改后代码
```tsx
<CardHeader>
  <div className="flex items-center justify-between gap-4">
    <CardTitle className="whitespace-nowrap">签署状态统计</CardTitle>
    <div className="w-64">
      <CompanySelector
        companies={companies}
        value={selectedCompanyId}
        onValueChange={setSelectedCompanyId}
      />
    </div>
  </div>
</CardHeader>
```

## CSS类名说明

### 外层flex容器
- `flex`：弹性布局
- `items-center`：垂直居中对齐
- `justify-between`：两端对齐
- `gap-4`：**新增**，元素之间间距16px

### CardTitle
- `whitespace-nowrap`：**新增**，禁止文字换行

### CompanySelector包裹容器
- `w-64`：**新增**，固定宽度256px

## 宽度选择说明

### 为什么选择w-64（256px）
1. **足够显示公司名称**：大多数公司名称在20个汉字以内，256px足够显示
2. **不会过宽**：不会占据过多空间，保持布局紧凑
3. **响应式友好**：在桌面端有足够空间，不会导致标题被挤压
4. **一致性**：与其他页面的选择器宽度保持一致

### 其他可选宽度
- `w-48`（192px）：更紧凑，但可能无法完整显示较长的公司名称
- `w-80`（320px）：更宽松，但会占据较多空间
- `w-64`（256px）：**推荐**，平衡了显示效果和空间占用

## 优势

1. **单行显示**：标题和选择器在一行显示，布局更紧凑
2. **视觉清晰**：标题和选择器之间有明确的间距，不会显得拥挤
3. **响应式友好**：固定宽度确保在不同屏幕尺寸下都能正常显示
4. **用户体验好**：选择器宽度足够显示完整的公司名称
5. **一致性**：与其他页面的布局风格保持一致

## 注意事项
1. 如果公司名称特别长（超过20个汉字），可能会被截断并显示省略号
2. 在小屏幕设备上（如平板或手机），可能需要进一步优化响应式布局
3. `whitespace-nowrap`确保标题文字不会换行，保持布局稳定
4. `gap-4`提供了适当的间距，避免标题和选择器紧贴在一起
